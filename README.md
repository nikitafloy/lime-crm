## Что это?

Выполненное тестовое - сверстать макет CRM из Фигмы с модальным окном.
Использовать методологию БЭМ и компонентный подход, минимум повторяющегося кода.
Компоненты должны переиспользоваться.

## Стэк

React, Webpack
